Уважаемые участники сообщества!

Разрешите мне поделиться еще одним докладом о проделанной командой Dash работе. Ниже приведен отчет о выполненной работе по состоянию на конец марта, который сгруппирован по четырем стратегическим направлениям, определенным на 2016 год.

1. Разработка программного обеспечения

1. Выпуск новой версии кошелька Dash - 12.x (ведущий разработчик: Эван Даффилд)

1. Находится в процессе разработки
2. Версия 12.1 исправит существующие проблемы управления (контракты, контроль)
3. Эван разработал новый план развития с пошаговыми указаниями (будет опубликован в ближайшее время)

2. Web-кошелек Dash-Эволюция (ведущий разработчик: Энди Фриер (Andy Freer)).

1. Командой достигнуты значительные успехи в разработке программного дизайна - большинство технологий уже были отобраны и протестированы. Приведено много доказательств успешности концепции кошелька.
2. По каждой проблеме разработки кошелька командой была создана доска интерфейсных задач на основе ПО JIRA.
3. Созданы колонки с пользовательскими историями и колонки с задачами, над которыми в настоящий момент идет работа.
4. Завершена разработка Skelton фреймворка для SPA и acidburn будет совершенствовать его.
5. Был завершен ранний прототип для демонстрации новой архитектуры, который показывает, что выбранные технологии будут нормально взаимодействовать друг с другом.
3. Dash-кошелек для Android - обновленные функции (ведущий разработчик: HashEngineering):

1. HashEngineering исправил многочисленные проблемы, связанные с основной частью кошелька
2. tomasz.ludek улучшил внешний вид и “юзабилити” кошелька.
3. Производится тестирование кошелька на разных устройствах и разных версиях Android
4. Dash-кошелек для iPhone

1. Выполняется работа над новым дизайном кошелька
2. Поскольку имя бренда Dash было изменено после того, как официальная версия кошелька была опубликована, команда собирается повторно утвердить кошелек с новым брендом на Apple Store
5. Разработка кошелька Electrum-Dash (внешний поставщик: Mazaclub)

1. Выполнена оценка объема работы
2. Работа началась в тесном сотрудничестве с основной командой Dash-разработчиков
3. Поддержка функциональных операций Мастернод должна быть завершена к концу первой недели мая.
4. Dash сеть начала поддерживать работу серверных операций Electrum-Dash
6. Интеграция с Lamassu (внешний поставщик: Deginner)

1. Завершение проекта значительно отстает от намеченного графика
2. Разработка альфа-версии кошелька завершена
3. Опубликовано видео с операциями Dash, выполняемыми на Lamassu-криптобанкомате по ссылке:

4. Выполняется работа по совершенствованию и исправлению ошибок в коде
7. Armory-кошелек (ведущий разработчик: moocowmoo)

1. Работа над проектом отложена до тех пока moocowmoo закончит работу над новым официальным форумом Dash
2. Развитие бизнеса

1. Запуск проекта фиатных шлюзов: https://forum.bits.media/index.php?/blog/178/entry-421-fiatnye-shliuzy-dash-postavleny-v-prioritet/
Проект состоит из следующих частей:

1. Фиатный шлюз (CryptoCapital.co)
2. Интерграция с Coinapult
3. Разработка back-end инструментов с открытым исходным кодом
4. Разработка следующих инструментов для Dash с открытым исходным кодом:

    • Кошелёк с мульти-подписью (Multi-sig) и хостинг сервисов совместных подписей (наподобие Deglet, Copay)
    • Сервис: брокеры (наподобие Coinapult, Shapeshift)
    • Сервис: биржа Покупки/Продажи DASH
    • Традиционные финансовые сервисы (наподобие Crypto Capital, Vogogo)
    • Сервисы Маркет-мейкеры и поставщики ликвидности

2. Осуществляется работа по интеграции с ПО для мгновенного обмена сообщениями Exodus

3. Установлен контакт с властями Свободной республики Либерленд

3. Маркетинг и коммуникации

1. Презентация Фернандо Гутьерреса (Fernando Gutierrez) на Фестивале интеренет-свободы https://dashtalk.org/threads/internet-freedom-festival-2016-report.8252/

2. Подкаст о Dash Фернандо Гутьерреса на сайте bitcoinsandgravy.com https://letstalkbitcoin.com/blog/post/episode-78-fernando-gutierrez-explains-dash

3. В рекламный графический пакет Dash добавлены новые кнопки для корпоративных продавцов: https://www.dash.org/promotional-graphics/

4. Запуск обновленного сайта dash.org

1. Реализация отстает относительно первоначального графика
2. Работа над структурой сайта находится на стадии разработки
3. Более подробный отчет можно посмотреть здесь: https://dashpay.atlassian.net/wiki/download/attachments/25002225/03.2016%20New-Website-Design-Project-Status-Report.pdf?api=v2

5. Новый форум на домене dash.org.

1. Тестовая версия форума запущена и работает (респект moocowmoo за невероятный вклад и отличную работу, проделанную в этой области)
2. В процессе проведение миграции контента из текущего форума
3. Решено большинство вопросов, связанных с работой плагина сайта
4. Организация проекта

1. Обновленное управление Dash с помощью шаблонов блокчейна: Шаблоны документов Системы Децентрализованного финансирования Dash (DGBB)

2. Запуск в работу фреймворка по управлению проектами Dash (фреймворк для контроля и помощи по надлежащему исполнению проектов)

Оригинальная версия отчета доступна по следующей ссылке: https://dashpay.atlassian.net/wiki/display/OC/March+2016+-+Dash+Core+Team+Monthly+Report
