Команда Dash рада анонсировать следующую стадию развития проекта:

Dash Evolution (Open Source).

Создатель и ведущий разработчик Dash, Эван Даффилд (Evan Duffield), объясняет главную цель “Эволюции”: "Она сделает Dash ещё более безопасной, позволит системе масштабироваться до невероятно больших размеров, и сделает Dash существенно проще в использовании, чем все остальные существующие криптовалюты.” Ключевой компонент новой системы - создание первого в мире Децентрализованного программного интерфейса для приложений (API). Эван продемонстрирует прототип этого DAPI на ежегодной Северо-Американской Bitcoin Конференции в Майями, которая пройдёт 21-22 Января 2016 года.

Интерфейсы API определяют то, как различные компьютерные программы и сети взаимодействуют друг с другом, являясь важным элементом, влияющим на взаимодействие пользователей с электронными устройствами. Недостатком всех имеющихся API, по словам Даффилда, является то, что: “API централизованы по своей природе. Все запросы к API производятся путём обращения к централизованному сервису. С внедрением Dash “Эволюции”, мы создаём новый тип API, взаимодействие с которым будет происходить полностью децентрализованным образом, но в то же время с высоким уровнем безопасности.” Децентрализованный API “Эволюции” позволит пользователям взаимодействовать с распределённой сетью Dash без необходимости скачивать и учиться использовать сложную программу-кошелёк, и без ожидания синхронизации своего кошелька с сетью. Главная цель проекта Dash “Эволюция” - сделать использование криптовалюты таким же простым, как использование PayPal и подобных сервисов, но без фактора централизации и без комиссий.

Dash Evolution - это масштабный проект и на полную его реализацию уйдут годы. Первую версию системы планируется реализовать в ближайшие 12-18 месяцев, в результате мы увидим Децентрализованный API (DAPI) и значительное увеличение масштабируемости (сотни тысяч операций в секунду станут возможны в сети DASH).

Команда Dash работает над “Эволюцией” в полностью открытом режиме. Вся документация и планы открыты для широкой публики и доступны на dash.org/evolution - будем рады получить замечания и предложения сообщества.
